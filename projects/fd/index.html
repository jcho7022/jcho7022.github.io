Node Failure Detector(Github)

Introduction

The server that we used to deploy our business application which is located in my dorm room kept on crashing. So to monitor for crashes I built a failure detector using Golang and placed the monitor program in a AWS free tier node(Robust node). Whenever, our server crashes the monitor program sends a email to me. This improved our servers up time.
