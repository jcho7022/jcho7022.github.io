Node Failure Detector(Github)

Introduction

The server me and my friend used to deploy and develop our business application kept on crashing. So to monitor detect crashes I built a failure detector using Golang and placed the monitor program in a AWS free tier node(Robust node) monitoring the server located in my dorm room. Whenever, our server crashes the monitor program sends a email to me. This improved server up time.
