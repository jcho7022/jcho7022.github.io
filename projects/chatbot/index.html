Chatbot & Android Messaging Application

Chatbot(Tensorflow, Python, XMPP, Docker)

How I Built It

I built the chatbot model following a Tutorial. The model is based on 170 million reddit comments that I pre-proccessed using python and sqlite. Training the model took 1 week with a GTX 1080ti. After training I put the model in a tensorflow docker container and used a python XMPP library to deploy it. When a request comes the text is inputed in the tensorflow library which outputs 30 suggested replies. Afterwards, a python library Levenshtein is used to score the replies and the program responds with the output with the highest score.

Messaging Service

Server(Node.js, MySql, XMPP(Ejabberd), Docker)
Android App(Java, Sqlite, XMPP(Smack))

What I built

A real time messaging service that enforces strict data consistancy over multiple devices.

Message Syncing Between Devices using Message Archive, HTTP

How I Built It

Server

Using the Docker version of an XMPP framework Ejabberd I enabled the module archive messaging and connected it to a MySql database. Every time a message is passed in before delivering it to the recipient Ejabberd archives the message into the database. For archiving the dialog list I set up a trigger in the MySql database such that when a archive message is inserted the dialog list for the user is also updated.

Client

Coded an application such that when the client app connects to the server the application pulls the archive messages and caches it in the application's Sqlite database. When the user accesses the dialog activity the cached dialog list is displayed. When the user accesses the messages activity the application queries the server for archive messages and caches the queried messaged on the app. The caching mechanism makes the messages load faster leading to better user experience and less load on the server.

Message Carboning

Real time message syncing between devices that is logged in with the same user id. Enabled on the server using Ejabberds module for the server and Smack on the application.
