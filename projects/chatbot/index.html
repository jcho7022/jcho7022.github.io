How I built it

I built the chatbot model following a Tutorial. The model is based on 170 million reddit comments that I pre-proccessed using python and sqlite. Trainning the model took 1 whole week with a GTX 1080ti. I than put the trained dataset inside a tensorflow docker container and used a python XMPP library to deploy it.
